Auth

회원 가입

  • 마이타민 섭취 시간(mytaminHour, mytaminMin) 데이터는 필수 요청값이 아니다.

    • 유저가 '건너뛰기’를 선택했을 경우 email, password, nickname 값만 넣어 요청한다.
      (→ 이 경우 mytaminHour, mytaminMin의 초기값은 null)

    • 마이타민 섭취 시간 정보는 추후 관련 기능 구현 시 변경 가능성이 있다.

HTTP request

POST /auth/signup HTTP/1.1
Content-Type: application/json;charset=UTF-8
Content-Length: 150
Host: MY-TAMIN-URL

{
  "email" : "mytamin@naver.com",
  "password" : "password1234",
  "nickname" : "강철멘탈",
  "mytaminHour" : "22",
  "mytaminMin" : "00"
}

HTTP response

HTTP/1.1 201 Created
Content-Type: application/json;charset=UTF-8
Content-Length: 252

{
  "statusCode" : 201,
  "message" : "회원 가입",
  "data" : {
    "email" : "mytamin@naver.com",
    "nickname" : "강철멘탈",
    "profileImgUrl" : "",
    "beMyMessage" : "",
    "mytaminHour" : "22",
    "mytaminMin" : "00"
  }
}

Exception Response

  • 회원 가입 요청 전 아래의 예외 상황들에 대해 미리 처리가 되었다고 가정한다. (가급적 발생하지 않아야 할 오류)

  • EMAIL_PATTERN_ERROR : 이메일 형식이 정해진 패턴("^(.[_a-z0-9-])*@(?:\w+\.)\w$")과 다를 경우

  • PASSWORD_PATTERN_ERROR : 비밀번호가 정해진 패턴("^(?=.*[0-9])(?=.*[A-Za-z]).{8,30}$")과 다를 경우

토큰 재발급

  • 새로 발급한 accessToken과 기존의 refreshToken을 리턴한다.

    • refreshToken의 만료 기간이 2일 이내로 남았을 경우 refreshToken 또한 새로 발급한다.

  • 만약 403 응답(= refreshToken 만료)일 경우에는 재로그인하도록 한다.

HTTP request

GET /auth/reissue HTTP/1.1
Content-Type: application/json;charset=UTF-8
Content-Length: 78
Host: MY-TAMIN-URL

{
  "email" : "mytamin@naver.com",
  "refreshToken" : "{{REFRESH_TOKEN}}"
}

HTTP response

HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
Content-Length: 164

{
  "statusCode" : 200,
  "message" : "토큰 재발급",
  "data" : {
    "accessToken" : "{{ACCESS_TOKEN}}",
    "refreshToken" : "{{REFRESH_TOKEN}}"
  }
}

    • am 5:00 ~ am 11:59 : 오늘도 힘차게 시작해볼까요 ?

Mytamin

  • takeAt 속성은 '마이타민 섭취 날짜’로, 행동 1 ~ 4가 일어나는 하루의 날짜 단위이다. (하루 진단 or 칭찬 처방 발생 시간을 뜻하는 것이 아님.)

하루 진단하기

  • 마음 컨디션은 String이 아닌, 각 마음 컨디션에 해당하는 int type의 코드를 보낸다.

  • mentalConditionCode

    • 1 : "매우 나빠요 .."

    • 2 : "나쁜 편이에요 :("

    • 3 : "그럭저럭이에요"

    • 4 : "좋은 편이에요 :)"

    • 5 : "매우 좋아요 !"

HTTP request

POST /mytamin/report HTTP/1.1
Content-Type: application/json;charset=UTF-8
X-AUTH-TOKEN: {{ACCESS_TOKEN}}
Content-Length: 158
Host: MY-TAMIN-URL

{
  "mentalConditionCode" : 5,
  "feelingTag" : "신나는",
  "todayReport" : "아무래도 아침형 인간이 되는건 너무 어려운 것 같다."
}

HTTP response

HTTP/1.1 201 Created
Content-Type: application/json;charset=UTF-8
Content-Length: 293

{
  "statusCode" : 201,
  "message" : "하루 진단하기",
  "data" : {
    "takeAt" : "2022.09.27.Tue",
    "mentalCondition" : "매우 좋아요 !",
    "feelingTag" : "신나는",
    "todayReport" : "아무래도 아침형 인간이 되는건 너무 어려운 것 같다."
  }
}
