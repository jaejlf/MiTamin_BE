# 🎉 프로젝트 생성
# ✨ feature : 새로운 기능 추가
# ➕ add : 부수적인 코드 추가 및 라이브러리 추가
# ✏️ update : 내부 코드의 유의미한 수정
# 🗑️ del : 쓸모없는 코드나 파일 삭제
# 🚚 move : 프로젝트 내 파일이나 코드의 이동
# 🐛 fix : 버그 수정
# 📝 docs : 문서 추가/수정
# ✅ test : 테스트 코드 관련
# ♻️ refactor : 코드 리팩토링
# 🚀 deploy : 배포와 관련된 작업
# 🙈 ignore : .gitignore 파일 수정
# 🎨 style : 코드 의미에 영향을 주지 않는 변경사항
# 🤝 merge : 다른 브랜치 MERGE 시 사용
# ⏪️ revert : 이전 버전으로 되돌리기
